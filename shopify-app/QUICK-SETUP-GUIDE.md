# ⚡ ClothyMax Quick Setup Guide (30-45 minutes)

## Free Product Images Sources
Get beautiful free fashion images from:
- **Unsplash**: https://unsplash.com/s/photos/fashion
- **Pexels**: https://www.pexels.com/search/fashion/
- **Burst by Shopify**: https://burst.shopify.com/fashion

---

## Step 1: Add Custom CSS (5 min)
1. Go to: **Online Store → Themes → Customize**
2. Click **Theme Settings** (bottom left gear icon)
3. Scroll to **Custom CSS**
4. Paste the entire code from `dawn-custom.css`
5. Save

---

## Step 2: Theme Settings (10 min)

### Colors (Theme Settings → Colors)
```
Background: #FFFFFF
Text: #121212
Buttons: #000000
Button text: #FFFFFF
Accent: #000000
```

### Typography (Theme Settings → Typography)
```
Headings: Assistant or any clean font
Body: Assistant
```

### Logo (Header → Logo)
- Use text logo: "CLOTHYMAX" 
- Or upload simple logo image

### Announcement Bar
```
Text: "FREE SHIPPING ON ORDERS OVER $50 ✨"
Background: #000000
Text color: #FFFFFF
```

### To Add a Product:
1. Admin → Products → Add product
2. Title: "Classic White T-Shirt"
3. Description: "Premium cotton blend for everyday comfort. Relaxed fit, soft fabric."
4. Price: $29
5. Upload 2-3 images from Unsplash/Pexels
6. Add sizes: S, M, L, XL
7. Save

---

## Step 4: Create Collections (5 min)

Create these collections:
1. **All Products** (automated: all products)
2. **New Arrivals** (manual)
3. **Best Sellers** (manual)
4. **Sale** (manual, add sale prices)

---

## Step 5: Homepage Sections (10 min)

In Theme Customizer, add sections in this order:

### 1. Image Banner (Hero)
- Use a fashion hero image from Unsplash
- Heading: "ELEVATE YOUR STYLE"
- Button: "SHOP NOW" → link to /collections/all

### 2. Featured Collection
- Collection: "New Arrivals"
- Products to show: 4-8

### 3. Image with Text
- Image: lifestyle fashion photo
- Heading: "Quality Meets Style"
- Text: "Curated pieces for the modern wardrobe"
- Button: "EXPLORE" → /collections/all

### 4. Featured Collection (again)
- Collection: "Best Sellers"
- Products: 4

---

## Step 6: Navigation (5 min)

### Main Menu:
- Shop All → /collections/all
- New Arrivals → /collections/new-arrivals
- Best Sellers → /collections/best-sellers
- Sale → /collections/sale

- Contact
